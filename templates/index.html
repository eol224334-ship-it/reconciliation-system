飞书通知配置

{{ feishuConfig.feishu_enabled ? '已启用' : '未启用' }}
开启后，收款/付款等操作会自动推送到飞书群 保存配置 发送测试消息 推送今日汇总
飞书机器人配置指南：
1. 打开飞书群聊 -> 点击群设置 -> 群机器人 -> 添加机器人 -> 选择「自定义机器人」
2. 填写机器人名称和描述，安全设置可选择「签名校验」
3. 复制 Webhook URL 填入上方输入框
4. 如果开启了签名校验，将密钥也填入上方「签名密钥」
5. 点击「保存配置」然后「发送测试消息」验证是否成功

通知触发事件：
- 新增对账流水 -> 蓝色卡片通知
- 客户收款 -> 绿色卡片通知
- 货值付款 -> 橙色卡片通知
- 佣金付款 -> 紫色卡片通知
- 手动推送汇总 -> 蓝色卡片通知（含各币种统计）
